The CSS Box Model

The CSS box model identifies the four parts of the HTML element that can be styled. Let's consider each part and see their effect.

The CSS Box Model is essential for understanding how elements on a webpage are displayed and interact with one another. It consists of four parts: content, padding, border, and margin. Below, we explore each of these parts and demonstrate how they affect an elementâ€™s layout.

The Four Parts of the CSS Box Model

  1. Content
  2. Padding
  3. Border
  4. Margin

Each HTML element has a content box, which contains the content. The size of the content box can be adjusted with CSS width and height properties. Surrounding the content are padding, borders, and margins, each of which can be styled individually using CSS.

Box Part CSS Properties Link to Reference
Content width, height none
Padding padding, padding-top, padding-right, padding-bottom, padding-left w3schools CSS Padding
Border border, border-style, border-width, border-color w3schools CSS Borders
Margin margin, margin-top, margin-right, margin-bottom, margin-left w3schools CSS Margins
